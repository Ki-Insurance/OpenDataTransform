# OasisDataConverter

Converts between one data source to another.

Full docs can be found [here](./docs).

## Dependencies

### System

python 3.8+

## Setup

To install the latest development version run:

```
pip install git+https://github.com/OasisLMF/OasisDataConverter.git
```
