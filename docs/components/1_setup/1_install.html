Installation¶

This section will get you started with transforming exposure data in the Open Data Transformation Framework.

There are two options for running transformations: using the command line or via the user interface.

Pre-requisites¶

To use the ODTF you will require:

  • Python version 3.8 or higher. To install Python, follow instructions at the Python For Beginners page.

  • To install the package via command line, you will require: the package pip.

It is recommended to install the package in a clean virtual environment.

Download and install¶

The tool operates on a local installation only. There are two options to install and run the tool.

Option 1: Installing the package via command line (Mac OS, Linux, Windows)

  1. Download the Python package from the project GitHub pages at github.com/OasisLMF/OpenDataTransform.

  2. Install the Python package using pip install git+https://github.com/OasisLMF/OpenDataTransform.git

Option 1: Install via the Windows executable file (Linux, Windows)

  1. Download the .exe file from at github.com/OasisLMF/OpenDataTransform.

  2. Run the .exe file
